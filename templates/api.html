Open API v0.1

Common resource information

  • Host: http://api.popong.com/
  • Version: v0.1
  • HTTP methods: GET
  • Response formats: json

Parameters

Options with astericks are default values.

Common

Every query should contain an API key. Now the only available API key is `test`.

  • api_key: [Issued API key] (Since we don't have registration page yet, please use `test`)

List

  • sort: [column name]
  • order: asc, desc*
  • page: [page number] (default: 1)
  • per_page: [number of items to be returned per request] (default: 20)

Search

  • q (mandatory): [query]
  • sort: [column name]
  • order: asc, desc*
  • page: [page number] (default: 1)
  • per_page: [number of items to be returned per request] (default: 20)
